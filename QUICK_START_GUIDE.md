# Quick Start Guide - AI Trading Strategy Workflow

## üöÄ Get Started in 5 Minutes

### Step 1: Import Workflow
1. Open your n8n instance
2. Go to **Workflows** ‚Üí **Import from File**
3. Select `ai-trading-strategy-workflow.json`
4. Click **Import**

### Step 2: Set Up YFinance API Server

**Option A: Use Docker (Recommended)**
```bash
cd /Users/alanphilip/Downloads/yfinance
docker-compose up -d
```

**Option B: Run Locally**
```bash
cd /Users/alanphilip/Downloads/yfinance
pip install -r requirements-api.txt
python api_server.py
```

Verify it's running:
```bash
curl http://localhost:8000/ticker/AAPL/price
```

### Step 3: Configure API URL in Workflow
1. Open the workflow in n8n
2. Click on **"Fetch Historical Stock Data"** node
3. Update the URL:
   - If using Docker: `http://localhost:8000/download`
   - If using remote server: `http://your-domain:8000/download`
4. Save the node

### Step 4: Add OpenAI Credentials
1. Click on any **OpenAI** node (AI Strategy Generator)
2. Click **Credential for OpenAI** dropdown
3. Select **Create New**
4. Enter your OpenAI API Key (get one at https://platform.openai.com/api-keys)
5. Click **Save**

### Step 5: Set Up Google Sheets
1. Go to Google Sheets: https://sheets.google.com
2. Create a new spreadsheet named: **"Strategy Performance Tracker"**
3. Add headers in Row 1:
   ```
   timestamp | strategy_name | tickers | total_trades | win_rate | total_return | max_drawdown | sharpe_ratio | market_analysis | recommendation
   ```
4. In n8n, click on **"Load Previous Strategy Performance"** node
5. Set up Google Sheets credentials:
   - Select **Create New Credential**
   - Choose **Service Account** authentication
   - Follow n8n's Google Sheets setup guide
6. Select your spreadsheet
7. Repeat for **"Save Performance Data"** node

### Step 6: Test the Workflow
1. Click **"Execute Workflow"** button in n8n
2. Wait 2-5 minutes for completion
3. Check results:
   - View output in **"Format Final Report"** node
   - Check Google Sheets for saved data

### Step 7: Activate for Weekly Runs
1. Toggle the **Active** switch (top-right corner)
2. Workflow will now run every Monday at 9:00 AM automatically

## üÜò Troubleshooting

### YFinance API Not Responding
```bash
# Check if server is running
curl http://localhost:8000/health

# Restart Docker container
docker-compose restart

# Or restart Python server
python api_server.py
```

### OpenAI API Errors
- **401 Unauthorized:** Check API key is correct
- **429 Rate Limit:** Wait or upgrade OpenAI plan
- **403 Forbidden:** Ensure GPT-4 access is enabled

### Google Sheets Access Denied
1. Make sure spreadsheet is shared with service account email
2. Service account email format: `xxx@xxx.iam.gserviceaccount.com`
3. Give "Editor" permissions

### Workflow Not Triggering
1. Check workflow is **Active** (toggle in top-right)
2. Verify timezone matches your location
3. Check n8n logs for errors
